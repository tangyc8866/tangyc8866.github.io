## 使用教程

1.  找到对应文件地址，点击右上角：edit

<img src="https://github.com/user-attachments/assets/92dfad50-e50a-4af2-8960-71e92b13fbfe" alt="image" width="965"/>

2.  添加/修改完毕后，点击右上方的 Commit changes（绿色）

3.  默认情况下，点击 Commit changes 即可

<img src="https://github.com/user-attachments/assets/272a9324-44ba-4325-8859-dcc65898c23b" alt="image" width="483"/>

4.  回到[主页](https://github.com/tangyc8866/tangyc8866.github.io/tree/main)，等待左上角的状态变为【✅】即可：

![](images/clipboard-3824868998.png)

5.  刷新[个人网站](https://tangyc8866.github.io/)，得到最新版本。
